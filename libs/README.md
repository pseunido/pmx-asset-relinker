Place required vendor DLLs here (do NOT commit them).

Expected layout:
 - libs/PEPlugin/PEPlugin.dll
 - libs/SlimDX/x64/SlimDX.dll

How to obtain:
 - Copy PEPlugin.dll from your local PMXEditor installation (path varies).
 - Copy SlimDX.dll from the SlimDX installation bundled with PMXEditor (use x64 if appropriate).

This folder is gitignored; each developer must place these files locally before building.
