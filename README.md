# PMX Asset Relinker

PMXエディタのプラグインです。

モデルに使用されているすべてのテクスチャ・ファイル（tex, sphere, toon）を指定したフォルダに書き出し、モデル自身も出力フォルダに保存します。

モデルが置かれたフォルダを単純に複製することとは異なり、次のような特徴があります。

特徴：
- モデルが使用しているテクスチャ・ファイルのみを出力します。
- パスにプレフィックス（サブフォルダ名）を追加して、テクスチャをサブフォルダに出力できます。
- サブフォルダ名を指定する場合は、モデルのテクスチャ・パスもサブフォルダを含むように置換されます。


## インストール

`PMXAssetRelinker.dll` を PMXEditor のプラグイン・フォルダ（`_plugin/User`）にコピーしてください。


## 使い方

モデルを PMXEditor で開き、[編集] - [プラグイン] - [User] から [PMX Asset Relinker] を選択します。

フォームが表示されますので、以下の項目を指定して **実行** ボタンをクリックします：

- **出力フォルダ** : テクスチャおよび更新されたモデルを出力するフォルダを選択します。
- **サブフォルダ名** : テクスチャの書き出しを、出力フォルダのサブフォルダ内にします。空欄の場合はサブフォルダは作成されません。
- **実行** : 指定した出力フォルダにテクスチャを出力し、モデルのテクスチャパスを置換して、モデル自身も出力フォルダに保存します。


## 開発

Visual Studio でプロジェクトを開き、 `libs` フォルダに必要な DLL を配置してからビルドしてください。
詳細は [libs/README.md](libs/README.md) を参照してください。
